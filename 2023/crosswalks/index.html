Ontology Alignment Evaluation Initiative - OAEI-2023 Campaign

Crosswalks Data Schema Matching

This track aims at evaluating the ability of systems to deal with the schema metadata matching task, in particular, with a collection of crosswalks from fifteen research data schemas to Schema.org [1]

It is based on the work carried out by the RDA Research Metadata Schemas Working Group. The collection serve as a reference for data repositories when they develop their crosswalks, as well as an indication of semantic interoperability among the schemas.

Dataset

The dataset is composed of 15 source research metadata schemas that have been aligned to Schema.org. The source schemas include discipline agnostic schemas Dublin Core, Data Catalogue Vocabulary(DCAT), Data Catalogue Vocabulary - Application Profile (DCAT-AP), Registry Interchange Format - Collections and Services (RIF-CS), DataCite Schema, Dataverse; and discipline schemas ISO19115-1, EOSC/EDMI, Data Tag Suite (DATS), Bioschemas, B2FIND, Data Documentation Initiative (DDI), European Clinical Research Infrastructure Network (ECRIN), Space Physics Archive Search and Extract (SPASE); as well as CodeMeta for software.

For the complete list of schemas is available here

Evaluation modalities and criteria

Using as a reference the manually established correspondences [1], the evaluation here will be based on the well-know measures of precision, recall and F-measure.
