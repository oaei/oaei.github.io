Ontology Alignment Evaluation Initiative - OAEI-2023 Campaign

Results for OAEI 2023 - Pharmacogenomics Track

Matching systems

We evaluated the following systems that registered for the track:
  • logmap-bio-melt
  • logmap-kg-melt
  • logmap-lite-melt
  • matcha_complex
  • matcha

Alignment results

Unfortunately, none of these systems were successful in producing alignments between reified n-ary tuples. It should be noted that some of the systems output alignments between entities that are not the target n-ary tuples.
