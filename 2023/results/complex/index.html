Ontology Alignment Evaluation Initiative - OAEI-2021 Campaign

Complex track - Evaluation

General description

The complex track aims at evaluating the systems which can generate complex correspondences.

This track contains 2 datasets: Conference, Populated Conference.

The detailed description of each dataset can be found at the OAEI Complex track page

Participating systems

This track is not attracting many participants since several years. This year, MatchaC, LogMap and LogMapLite have been registered to participate. While LogMapLite and LogMap are dedicated to generate simple correspondences, only LogMap was able to generate alignments.

MatchaC, the only systems specifically designed to generated expressive alignments, had some problems to deal with the datasets and was not able to generate valid correspondences.

Conclusions

As last year, this track attracts a too few number of participants. Unfortunately, several datasets have also been discontinued (Hydrography, GeoLink, Populated GeoLink, Populated Enslaved and Taxon) given the unavailability of organisers. New organisers are welcome. So, several efforts have to be made in order to increase participation and gather also new track organisers.
